# legacy/ — ранние прототипы

Здесь лежат первые версии проекта. Они **не поддерживаются** и оставлены для справки.
Актуальный путь — `analyze.py` + `core/analyzer.py` в корне репозитория.

- `main.py` — десктоп-версия на OpenCV (два окна: видео + телеметрия, отчёт в matplotlib).
- `app.py` — веб-прототип на Streamlit (на момент архивации содержит баг: вызывает
  `ShotDetector.check_shot` с неверным числом аргументов).
- `testlib.py`, `check_mp.py`, `check_gpu.py` — отладочные скрипты проверки окружения.

Зависимости для запуска — в `legacy/requirements.txt`. Запускать из корня репозитория,
например: `python -m legacy.main` (может потребоваться доработка импортов).
