# 🏀 Shot Analyzer — анализатор баскетбольного броска

Инструмент для разбора техники броска по видео: распознаёт позу игрока,
считает углы (локоть, колено, стопы), детектирует фазы броска и собирает статистику.

> Проект в активной разработке. Текущий план развития — в [`ROADMAP.md`](ROADMAP.md).
> Сейчас завершён **Этап 0 (навести порядок)**: чистый headless-пайплайн «видео → метрики».

## Как это работает

- **MediaPipe Pose** распознаёт скелет человека на каждом кадре.
- По ключевым точкам считаются углы суставов (`core/detector.py`).
- Стейт-машина определяет фазы броска `IDLE → PREP → RELEASE` и собирает метрики
  по каждому броску (`core/state_machine.py`).
- Единый пайплайн `core/analyzer.py` прогоняет видео и возвращает структурированный результат.

## Установка

Нужен Python 3.10+. GPU не требуется.

```bash
python -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
```

## Запуск

```bash
python analyze.py path/to/shot.mp4
# сохранить результат в JSON:
python analyze.py path/to/shot.mp4 --json result.json
```

Пример вывода:

```
Кадров всего: 240
Кадров с распознанной позой: 232
Найдено бросков: 3

#  | elbow | knee_load | feet_diff | grade
--------------------------------------------------
1  | 168   | 132       | 8         | Great Arc
2  | 151   | 121       | 12        | OK
3  | 142   | 158       | 9         | Short Arm (No Legs)
```

Лучшие результаты — если снимать **сбоку** (виден угол локтя и колена).
Устойчивость к произвольному ракурсу появится на Этапе 1 (см. roadmap).

## Структура

  state_machine.py   # ShotDetector — фазы броска и метрики
  analyzer.py        # analyze_video() — единый пайплайн (источник правды)
